# 오늘의 해외 뉴스 앱

## 프로젝트 개요

**프로젝트명:** 오늘의 해외 뉴스  
**개발 언어:** React Native  
**백엔드:** Spring Boot, MySQL  

## 기능

1. 해외 뉴스 기사 제공
2. 카테고리별 인기 기사 제공
3. 주요 이슈 기사 요약 제공
4. 날씨 정보 제공
5. AI 번역
6. AI 요약
7. AI 키워드

## 기술 스택

**프론트엔드:**
- React Native

**백엔드:**
- Spring Boot
- MySQL
- JPA
- OpenAI
- LangChain

**기타:**
- GitHub (버전 관리 및 협업)
